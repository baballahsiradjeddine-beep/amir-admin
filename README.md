# Amir Nouadi Management System
## نظام إدارة الشركات والمعاملات المالية

تطبيق سطح مكتب شامل لإدارة الشركات، المزودين، والمعاملات المالية مع نظام صندوق رأس المال.

## المميزات الرئيسية

- ✅ **إدارة الشركات**: تتبع شركاتك ورأس المال العامل لكل شركة
- ✅ **إدارة المزودين**: إدارة قائمة المزودين والموردين
- ✅ **شركات العملة**: إدارة شركات الصرافة والمعاملات
- ✅ **صندوق رأس المال**: نظام مركزي لإدارة الرصيد المالي
- ✅ **النسخ الاحتياطي**: نسخ احتياطي تلقائي ويدوي للبيانات
- ✅ **التقارير والإحصائيات**: تحليلات مالية شاملة
- ✅ **وضع أوفلاين**: يعمل بالكامل بدون اتصال بالإنترنت

## متطلبات التشغيل

- Windows 10 أو أحدث
- macOS 10.13 أو أحدث
- Linux (Ubuntu 18.04+ أو ما يعادلها)

## التطوير

### تشغيل التطبيق في وضع التطوير

```bash
# تثبيت المكتبات
npm install

# تشغيل Next.js فقط
npm run dev

# تشغيل Electron مع Next.js
npm run electron:dev
```

### بناء التطبيق للإنتاج

```bash
# بناء Next.js كـ static export
npm run build

# بناء تطبيق Electron
npm run electron:build

# أو بناء ملف قابل للتوزيع
npm run dist
```

## البنية التقنية

- **Frontend**: Next.js 16 + React 19
- **UI**: Tailwind CSS + Radix UI
- **Database**: Better-SQLite3 (قاعدة بيانات محلية)
- **Desktop**: Electron 28
- **Icons**: Lucide React
- **Forms**: React Hook Form + Zod

## الملفات الهامة

- `electron/main.js` - نقطة الدخول الرئيسية لـ Electron
- `electron/preload.js` - سكريبت الأمان لـ Electron
- `lib/sqlite/` - طبقة قاعدة البيانات
- `app/` - صفحات ومكونات Next.js
