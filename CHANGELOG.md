# سجل التغييرات

جميع التغييرات الملحوظة في هذا المشروع سيتم توثيقها في هذا الملف.

## [1.0.0] - 2026-01-29

### أُضيف
- **المصادقة**: نظام دخول آمن مع بيانات اعتماد محمية
- **لوحة التحكم الرئيسية**: واجهة محترفة مع شريط جانبي للملاحة
- **المعاملات اليومية**:
  - إضافة معاملات مع دعم التعبيرات الحسابية
  - تحويل العملات مع معدل قابل للتعديل
  - عرض النتائج بتنسيق مخصص (فواصل ألوف)
  - تتبع جميع المعاملات مع التاريخ والوصف

- **إدارة الشركات**:
  - إضافة شركات جديدة
  - تتبع رأس المال الأساسي والفعلي
  - عرض الشركات كقوائم وبطاقات
  - الحسابات الديناميكية للأرصدة

- **إدارة المزودين**:
  - إضافة مزودين بعملات مختلفة (USD, RMB)
  - معاملات مستقلة عن رأس المال
  - حساب صرف العملات الأجنبية

- **الإحصائيات والتقارير**:
  - رسوم بيانية تفاعلية (شريطية، خطية)
  - إحصائيات دخل ومصروفات
  - حساب الأرباح والخسائر
  - ترتيب الشركات حسب الأداء

- **الإعدادات**:
  - تبديل بين الوضع الفاتح والداكن
  - إدارة بيانات الحساب (البريد وكلمة المرور)
  - حماية البيانات القديمة
  - خيارات الخصوصية

- **الميزات الإضافية**:
  - دعم كامل للغة العربية (RTL)
  - تصميم استجابي (Responsive)
  - إشعارات toast للعمليات
  - معالجة الأخطاء بشكل احترافي

### التكنولوجيا المستخدمة
- Next.js 16
- React 19.2
- TypeScript
- Tailwind CSS v4
- Shadcn/ui
- Recharts
- next-themes
- React Hook Form
